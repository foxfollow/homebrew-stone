# Homebrew Taps

![from macOS 15](https://img.shields.io/badge/macOS-15.0+-blue.svg) <br>

A Homebrew tap containing custom casks for macOS applications.

## Installation

### Homebrew (macOS)
```bash
brew install foxfollow/stone/stone-clipboarder-tool
```

### Manual Download
Download the latest `.zip` from [Releases](https://github.com/foxfollow/Stone-Clipboarder-Tool/releases)
